Gemini Desktop

Modern Desktop UI for Gemini CLI

A powerful, cross-platform desktop and web application built with Tauri and React. Seamlessly integrate with Gemini CLI for enhanced AI interactions.

Features

Cross-Platform Desktop

Native desktop app for Windows, macOS, and Linux built with Tauri

Web Version

Standalone web server with Rocket backend for browser access

Multiple Models

Support for Gemini 2.5 Pro, Flash, and Flash-Lite models

Syntax Highlighting

Beautiful markdown rendering with code syntax highlighting

Tool Integration

Advanced tool call support and MCP server management

Theme System

Dark/light themes with customizable interface

Built With

Rust + Tauri
React + TypeScript
Rocket Server
Vite + pnpm
